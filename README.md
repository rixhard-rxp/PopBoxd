PopBoxd - Registro y Crítica de Música Pop

Aplicación móvil nativa desarrollada con React Native y Expo para el registro, gestión y evaluación de álbumes musicales del género pop. 

Definición del problema y justificación del alcance de la aplicación: Crear un programa para registrar tus albums desde el código fuente para hacer Reviews de las canciones de cada album.

Requisitos previos del entorno de ejecución: 
Para el correcto despliegue del servidor de desarrollo, el entorno del evaluador debe contar con:
Node.js: Versión LTS activa (Recomendado v18.x o v20.x).
NPM: Incluido con la instalación de Node (v9.x o superior).
Expo CLI: Gestionado global o localmente a través de las herramientas nativas del empaquetador.
Dispositivo Físico: Dispositivo iOS o Android con la aplicación móvil Expo Go instalada para pruebas en vivo.

Siga de manera secuencial los siguientes comandos en la terminal para:

Clonar el repositorio:
1) bash
2) git clone 
3) <https://github.com/erozog125/portfolio-react-native.git>
4) cd PopBoxd
5) npm install
6) npx expo start -c

Correr el Proyecto:
1) npm run web 
2) Escanear el código QR
2) Presionar -W- para abrir el proyecto en un servidor local
